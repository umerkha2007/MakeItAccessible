### For Chrome/Edge/Brave

1. **Download or clone the repository:**
   ```bash
   git clone https://github.com/umerkha2007/MakeItAccessible.git
   ```
   
   Or download the ZIP file and extract it.

2. **Open your browser's extension page:**
   - **Chrome:** Navigate to `chrome://extensions/`
   - **Edge:** Navigate to `edge://extensions/`
   - **Brave:** Navigate to `brave://extensions/`

3. **Enable Developer Mode:**
   - Look for the "Developer mode" toggle in the top right corner
   - Click to enable it

4. **Load the extension:**
   - Click "Load unpacked" button
   - Navigate to the `MakeItAccessible` folder
   - Select the folder and click "Select Folder" (or "Open" on Mac)

5. **Verify installation:**
   - You should see the MakeItAccessible icon in your browser toolbar
   - The extension card should appear in your extensions list
   - Click the icon to open the popup and test features

## ğŸ› Troubleshooting

### Extension doesn't appear after loading
- Make sure you selected the correct folder (should contain `manifest.json`)
- Check for errors in the extension card
- Try reloading the extension

### Features not working on a page
- Some sites (chrome://, about:, file://) block extensions for security
- Refresh the page after enabling the extension
- Check browser console for errors (F12)

### Text-to-Speech not working
- Ensure your browser supports Web Speech API
- Check that your page is using HTTPS (not HTTP)
- Verify microphone permissions if prompted

### Settings not saving
- Check that the extension has storage permissions
- Try removing and reinstalling the extension
- Clear browser cache and reload
